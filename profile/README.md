# clipstash

cime 플랫폼 영상을 원하는 구간만 클립으로 저장하는 비공식 개인 도구 모음입니다.

<br>

## 프로젝트

| 레포 | 설명 |
|------|------|
| [cime-clip-server](https://github.com/clipstash/cime-clip-server) | 스트림 메타데이터 추출 및 CORS 프록시 백엔드 API |
| [cime-clip-app](https://github.com/clipstash/cime-clip-app) | 클립 구간 선택 및 브라우저 다운로드 프론트엔드 |

<br>
